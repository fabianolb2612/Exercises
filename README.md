🌐 Web Programming Exercises

This repository contains a collection of exercises and practical activities developed during my studies in Web Programming. The goal is to reinforce fundamental concepts and improve problem-solving skills using modern web technologies.

📚 About

The exercises included here cover a variety of topics related to web development, such as:

Object-Oriented Programming (OOP)

PHP fundamentals

Classes, methods, getters, and setters

Autoloading and namespaces

Basic front-end structure (HTML, CSS, JavaScript)

Problem-solving with real-world scenarios

Each exercise is designed to simulate common situations in web development, helping to build a strong foundation for creating scalable and maintainable applications.

📁 Project Structure

The repository is organized into folders by topic or exercise:

/source         -> Main application classes and models  
/public         -> Entry points and test files  
/exercises      -> Individual exercise implementations  

Note: The structure may vary slightly depending on the exercise.
